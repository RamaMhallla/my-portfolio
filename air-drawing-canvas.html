Air Drawing Canvas

Draw in the air with your index finger using a standard webcam. The app uses OpenCV for image processing and MediaPipe Hands for real-time hand tracking, then renders smooth strokes on a virtual canvas with a clean UI.

Overview

This app turns simple finger gestures into digital ink. It includes:

  • Side control to adjust brush thickness
  • Primary color buttons (Red, Green, Blue)
  • Eraser mode
  • Background toggle (white/black)
  • Grid overlay with adjustable spacing
  • One-key saving (canvas + preview)
  • Motion filtering to reduce accidental strokes

Key Features

  • Real-time index-finger tracking with smooth drawing
  • Left-side thickness slider (hover the index finger to adjust)
  • RGB color buttons and Eraser mode
  • Toggle background without affecting existing strokes
  • Grid overlay with G− / G+ spacing controls
  • Open-palm clear gesture + “Clear All” button
  • Background-subtraction guard to prevent unintended lines
  • On-screen color indicator showing current HEX
  • Quick save with s (drawing_*.png & preview_*.png)

How It Works

  1. Camera input: capture frames from the webcam
  2. Hand tracking: MediaPipe detects hand landmarks; index fingertip (ID 8) is tracked
  3. Gesture logic: “Index only up” → draw/erase; “Open palm” → clear
  4. Drawing engine: anti-aliased strokes on a separate canvas
  5. Controls: bottom toolbar (colors, eraser, grid, background, clear) + side thickness slider

Controls

Action How
Draw Raise index finger only and move within the drawing area
Eraser Tap Eraser then draw (uses background color with larger width)
Thickness Hover the side slider to adjust stroke width
Color Use color buttons (Red / Green / Blue)
Background Tap BG to toggle between white/black
Grid Tap G− / G+ to change spacing
Clear Open palm for a moment, or press Clear All
Save / Quit s to save (canvas + preview), q to quit

Requirements

  • Python 3.10+ and a webcam (Windows/macOS/Linux)
  • Packages: opencv-python, mediapipe, numpy
